Installation Readme

Installing an AppVet tool requires two main steps:
  • Step 1. Configure AppVet: AppVet must be configured to connect to and interact with the (online) tool. This is done by modifying the tool's adapter used by AppVet.
  • Step 2. Install and configure the tool: The tool must be configured to connect to and interact with AppVet. This is done by modifying the tool's ToolProperties.xml file.
IMPORTANT! If the tool already exists online and is accessible to AppVet, then AppVet should be configured to connect to that existing tool and Step 2 should be skipped.
